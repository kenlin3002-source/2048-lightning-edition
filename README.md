# ⚡ 2048 Lightning Edition (閃電進化版)

這是一款基於 C 語言與終端機介面開發的 2048 衍生益智遊戲。本專案不僅重現了經典的 2048 滑動合併核心玩法，更加入了**時間壓迫機制、特殊道具方塊、時空回溯系統，以及全彩無閃爍的大色塊 UI 介面**，大幅提升了遊戲的節奏感與策略深度。

> **開發環境**：C Language / GCC Compiler
> **終端機引擎**：`rlutil.h` (跨平台終端機色彩與游標控制)

---

## 🎮 核心創新玩法 (Features)

本專案跳脫傳統 2048 的靜態框架，實作了以下進階功能：

* **⏱️ 動態閃電計時器 (Timed Combo System)**
  * 支援開局難度選擇：極限 (20秒)、標準 (45秒)、休閒 (60秒)。
  * 畫面上方具有動態圖形進度條 `[====  ]`。成功合併方塊可獲得時間回補（Combo Bonus），時間歸零即判定 Game Over。
* **⏪ 時空回溯系統 (Undo Stack)**
  * 底層利用「結構體陣列 (Struct Array)」實作 Stack 堆疊概念。
  * 允許玩家按下 `U` 鍵消耗 50 分回到上一步，最高可追溯 50 步歷史紀錄。
* **🌟 稀有特殊方塊 (Special Tiles)**
  * **黃金方塊 `[*N*]`** (機率 10%)：合併後分數 3 倍，並大幅回補 5 秒時間。
  * **百搭史萊姆 `[ ? ]`** (機率 5%)：以紫色呈現，可以與**任何數字**進行無條件合併並使其翻倍，突破死局的神器！
* **💾 最高紀錄持久化 (File I/O)**
  * 透過 C 語言的 `fopen` 讀寫功能，遊戲會自動產生 `record.txt`。
  * 即使關閉遊戲，你的最高分 (PR) 依然會被永久保存並顯示於畫面中。

## 💻 技術亮點 (Technical Highlights)

1. **無閃爍渲染技術 (Zero-Flicker Rendering)**：
   捨棄傳統 `system("clear")` 導致的畫面閃爍，全面採用 `rlutil.h` 的 `locate(x, y)` 絕對座標覆蓋技術，並繪製 $3 \times 7$ 的大型實體色塊，達到現代遊戲般的流暢視覺體驗。
2. **一維陣列降維打擊 (1D Array Sliding Algorithm)**：
   在處理 WASD 四個方向的移動時，透過指標與迴圈將二維矩陣的行列「抽取出成為一維陣列」，進行統一的「靠攏 $\rightarrow$ 合併 $\rightarrow$ 靠攏」邏輯處理後再塞回，程式碼極度精簡且無冗餘。
3. **防呆與按鍵緩衝區處理**：
   自訂 `handleGameOver` 結算畫面，攔截多餘的鍵盤緩衝輸入，並引入超大型 ASCII Art 視覺特效（勝利/失敗具有不同演出）。
